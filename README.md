## file structure
```
backend:
	database/
		models.py 	| 
		db.py		| 
	routers/  		| REST-API-endpoints
	auth/	  		| login, auth

	main.py	  		| main entrypoint.
	
frontend:
	routes/	  | the different pages
	lib/ 	  | reusable components
```

## dev
for developing I recomend doing npm install inside /frontend
